2. DigramDartCount.png

Create a technical scoring diagram for a dartboard hit in normalized coordinates for software documentation. Show a normalized 800x800 dartboard view with center near (398, 400) and board radius 300. Mark one example hit point and visualize how score is computed from both radial distance and angular sector.

Show these exact scoring bands as labeled concentric rings based on board radius:
- BULL: distance <= 5% of radius
- OUTER_BULL: 5% to 9%
- TRIPLE ring: 53% to 59%
- DOUBLE ring: 88% to 95%
- MISS: outside 95%
- all remaining in-board regions are SINGLE

Also show that sector is computed from the angle of the vector from center to hit point, using 20 sectors of 18 degrees each. Add a note that the angle comes from atan2(-dy, dx), then optional angle offset, then a rotation offset of 3 sectors. Show sector numbering in classic dartboard order:
20, 5, 12, 9, 14, 11, 8, 16, 7, 19, 3, 17, 2, 15, 10, 6, 13, 4, 18, 1

Include a worked example label in the diagram like:
hit -> distance -> ring
hit -> angle -> sector
ring + sector -> final score

Style: clean flat vector infographic, engineering documentation style, white background, thin precise lines, green for bull rings, blue for triple ring, red for double ring, orange highlighted hit vector, readable labels, no extra visual noise.
